# Product Billing System

A simple Java AWT-based billing application for managing products and calculating totals with tax.

## Features
- Add products
- Tax calculation
- Grand total calculation
- Input validation
- GUI interface using Java AWT

## Technologies Used
- Java
- AWT
- OOP
- Event Handling

## How to Run

1. Clone the repository
2. Open project in IntelliJ or Eclipse
3. Run `BillingSystem.java`
